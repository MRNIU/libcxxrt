## 使用方法 Usage

```shell
git clone https://github.com/MRNIU/MiniCRT.git
cd MiniCRT
mkdir build
cmake ..
make
```

构建完成后，库文件保存在 build/lib/libminicrt.a
测试程序为 build/bin/test
